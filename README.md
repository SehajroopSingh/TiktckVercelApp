# ContentLab

A simple web-based tool for creating and testing educational media content with user control.

## Overview

ContentLab is built with Next.js (App Router), TypeScript, and Tailwind CSS. It is designed to be a lightweight, production-ready landing page for app review requirements and early product introduction.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This app is ready to be deployed on [Vercel](https://vercel.com).

1. Install Vercel CLI:
   ```bash
   npm install -g vercel
   ```

2. Deploy:
   ```bash
   vercel
   ```

3. Connect your custom domain `contentlab.app` in the Vercel project settings.
